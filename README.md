## How to run this

### Backend

```bash
# install dependencies
$ npm install

# start the server
$ npm run start
```

### Frontend

Please keep both Backend and Frontend running if you're trying this out.

```bash
# install dependencies
$ npm install

# start dev server
$ npm run dev
```
